Chat-App 💬

A real-time full stack chat application built using the MERN Stack with private messaging, group chats, online status, typing indicator, file sharing, emojis, and Socket.IO real-time communication.

🚀 Features
✅ Authentication,
User Register & Login,
JWT Authentication,
Protected Routes.
✅ Real-Time Chat,
One-to-One Messaging,
Real-Time Messages using Socket.IO,
Online/Offline Users,
Typing Indicator,
✅ Group Chat,
Create Groups,
Send Group Messages,
Real-Time Group Communication,
✅ Media Sharing,
Send Images,

⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/AyushSE27/chat-app.git
2️⃣ Backend Setup
cd backend
npm install
Create .env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
Start Backend
node server.js
3️⃣ Frontend Setup
cd frontend
npm install
npm run dev
